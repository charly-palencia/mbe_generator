# mbe generator

How it works
=====
*Download project*

     git clone https://github.com/charly-palencia/mbe_generator.git && cd mbe_generator
     
*Install Dependencies*

    bundle install && rake mbe:create_database
    
Commands
====
*Create user*

    rake mbe:create_user

*Remove user*

    rake mbe:remove_user

*Generate file* 

Run mbe:create_user too!

    rake mbe:generate
   

Enjoy your MBE File :)
===
